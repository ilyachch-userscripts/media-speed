# Media Speed

A userscript that allows you to change the playback speed of media (video and audio) on any website.

## Features

- **Speed Control**: Easily change playback speed using an overlay control panel.
- **Customizable Options**: Set global or site-specific speed presets.
- **Keyboard Shortcuts**:
    - \`Ctrl\` + \`Alt\` + \`.\` : Increase speed
    - \`Ctrl\` + \`Alt\` + \`,\` : Decrease speed
    - \`Shift\` + \`ArrowRight\` / \`ArrowLeft\` : Seek (Large step)
    - \`Ctrl\` + \`ArrowRight\` / \`ArrowLeft\` : Seek (Medium step)
- **Visual Feedback**: Shows a notification when speed is changed.
- **Persistent Settings**: Remembers your speed preferences.

## Installation

1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/).
2. Click [here](https://raw.githubusercontent.com/ilyachch-userscripts/media-speed/main/media-speed.user.js) to install the script.

## Usage

1. Visit any website with video or audio content.
2. A speed control panel will appear in the bottom left corner when media starts playing or on hover.
3. Click on a speed option to set the playback speed.
4. Use the userscript menu to configure global or site-specific speed options.
